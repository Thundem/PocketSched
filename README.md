# PocketSched 📱

Швидкий мобільний застосунок для розкладу університетських занять із ручним введенням та OCR-сканером таблиць розкладу. Створений як зручна альтернатива громіздким університетським додаткам.

## ✨ Особливості

- **Локальне зберігання:** Всі дані зберігаються у SQLite на пристрої — без реєстрації та серверів.
- **Темна тема:** Мінімалістичний дизайн, тільки важлива інформація.
- **Навігація по днях:** Вкладки "Сьогодні", "Завтра", "Тиждень" зі свайпом.
- **Time Engine:**
  - Автоматично визначає чисельник (непарний) та знаменник (парний) тиждень.
  - Підсвічує пару, що триває прямо зараз.
- **Ручне керування парами:** Додавання, редагування та видалення через зручний інтерфейс.
- **OCR-сканер розкладу:**
  - Хмарне розпізнавання через [OCR.space API](https://ocr.space/) з підтримкою кирилиці.
  - Розпізнає формат таблиці: номер пари, час, назва предмету, викладач, аудиторія.
  - Автоматично визначає день тижня з тексту фото.
  - Розпізнає підгрупу (`підгр. 1/2`) та півпару (`1 півпара / 2 півпара`).
  - Ігнорує порожні слоти (пари без предмету).
  - Зупиняє парсинг при переході на наступний день у таблиці.
- **Деталі пари:** Натисніть на картку — відкриється нижня панель з повною інформацією та кнопками редагування/видалення.
- **Скорочення ПІБ:** У картці відображається "Прізвище І.П.", у деталях — повне ім'я.
- **Кастомний кроп:** Перед скануванням можна вибрати потрібну область фото.

## 🛠 Технологічний стек

| Інструмент | Призначення |
|---|---|
| [React Native](https://reactnative.dev/) + [Expo SDK 55](https://expo.dev/) | Основний фреймворк |
| TypeScript | Мова розробки |
| [Expo SQLite](https://docs.expo.dev/versions/latest/sdk/sqlite/) | Локальна база даних |
| [React Navigation](https://reactnavigation.org/) | Bottom Tabs, Material Top Tabs, Native Stack |
| [OCR.space API](https://ocr.space/) | Хмарне розпізнавання тексту (кирилиця) |
| [expo-file-system](https://docs.expo.dev/versions/latest/sdk/filesystem/) | Читання зображення у Base64 |
| [expo-image-picker](https://docs.expo.dev/versions/latest/sdk/imagepicker/) | Вибір фото з галереї |
| Expo Vector Icons (`Ionicons`) | Іконки |

> **Важливо:** Застосунок використовує нативні модулі і потребує **EAS Development Build** — Expo Go не підтримується.

## 🚀 Запуск локально

### Передумови

- [Node.js](https://nodejs.org/) (рекомендовано LTS)
- [EAS CLI](https://docs.expo.dev/eas-update/getting-started/): `npm install -g eas-cli`
- Встановлений dev build на Android-пристрої (зібраний через `eas build --profile development`)

### Інструкція

1. **Встановіть залежності:**
   ```bash
   npm install
   ```

2. **Запустіть Metro Bundler:**
   ```bash
   npx expo start --dev-client
   ```

3. **Підключіться з телефону:**
   Відскануйте QR-код у застосунку **Expo Dev Client** (не Expo Go).
