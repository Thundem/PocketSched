# PocketSched 📱

Швидкий, повністю офлайновий застосунок для розкладу занять із ручним введенням та розумним OCR-сканером. Створений як незалежна та надійна альтернатива складним університетським додаткам (таким як Unisched).

## ✨ Особливості

- **Повний Офлайн:** Всі дані зберігаються локально на вашому пристрої за допомогою бази даних (SQLite). Для роботи не потрібен інтернет.
- **Мінімалістичний Дизайн:** Темна тема, яка не напружує очі, і тільки найважливіша інформація на екрані.
- **Розумна Навігація:** Верхні свайпи між днями ("Сьогодні", "Завтра", "Тиждень"). 
- **Time Engine (Калькулятор часу):**
  - Автоматично розпізнає "Чисельник" (непарний) та "Знаменник" (парний) тиждень.
  - Динамічно підсвічує пару, яка триває прямо зараз.
- **Ручне керування:** Додавайте предмети вручну через інтуїтивно зрозумілий застосунок із зручними перемикачами (Segmented Controls).
- **OCR Сканер (У розробці):** Можливість імпортувати розклад, просто підвантаживши його скріншот чи фотографію. Вбудований алгоритм парсить текст і перетворює його на картки розкладу.

## 🛠 Технологічний стек

* **Фреймворк:** [React Native](https://reactnative.dev/)
* **Середовище:** [Expo](https://expo.dev/) (профіль: Blank TypeScript)
* **Мова:** TypeScript
* **Навігація:** React Navigation (Bottom Tabs, Material Top Tabs, Native Stack)
* **База даних:** [Expo SQLite](https://docs.expo.dev/versions/latest/sdk/sqlite/)
* **Іконки:** Expo Vector Icons (`Ionicons`)

## 🚀 Як запустити проект локально

### Передумови
Переконайтесь, що у вас встановлені [Node.js](https://nodejs.org/) та Git. Також рекомендується встановити програму **Expo Go** на ваш смартфон (доступно в App Store та Google Play).

### Інструкція з інсталяції

1. **Клонуйте репозиторій або перейдіть до папки проекту:**
   ```bash
   cd PocketSched
   ```

2. **Встановіть залежності:**
   ```bash
   npm install
   ```

3. **Запустіть середовище розробки Metro Bundler:**
   ```bash
   npm run start
   ```

4. **Тестування:**
   - Після запуску `npm run start` у терміналі з'явиться QR-код.
   - Відкрийте застосунок **Expo Go** на своєму смартфоні і відскануйте цей QR-код.
   - Або натисніть клавішу `a`, щоб відкрити на емуляторі Android (якщо він встановлений і запущений), або `i` для iOS (тільки на macOS).
